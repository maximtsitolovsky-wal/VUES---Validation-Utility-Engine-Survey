⚡ Code Puppy Decision Protocol
  1. 1. Read MEMORY.md — skip thinking on settled decisions
  2. 2. Check skills/INDEX.md — apply existing skill if pattern matches
  3. 3. Execute task using settled context
  4. 4. Write new decision to MEMORY.md
  5. 5. Extract reusable skill to skills/ if pattern will repeat
Less thinking. More decisions. Memory + Skills = autonomy.

📧 Email Gate Logic
● SMTP_ENABLED=true — Python sends email directly via emailer.py
● SMTP_ENABLED=false (default) — Python writes status to Airtable field. Airtable automation rule watches that field and sends the email.
⚠️ Business rule: NEVER write ERROR to Airtable. Pipeline exception → FAIL with score 0.
